**AI Infrastructure** — OpenClaw agent setup, n8n workflow automation, multi-agent systems. Self-hosted, hardened, monitored.

**Server Security** — Linux hardening, incident response, WordPress/WooCommerce security. 15+ years keeping production systems alive.

**Automation Rescue** — Your workflow broke. Your API tokens expired. Your agent went silent. I fix it and make sure it doesn't happen again.
